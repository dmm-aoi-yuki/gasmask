- hostsファイルの書き込みを特権付き cp 方式に変更し、権限まわりの安定性を向上
- SF Symbol メニューアイコンの描画を簡略化

## 動作環境

- macOS 13 Ventura 以降
- Apple Silicon Mac

## インストール

### 「壊れているため開けません」と表示される場合

本アプリは Apple Developer ID による署名・公証を行っていないため、macOS Gatekeeper によってブロックされることがあります。

**方法 1: ターミナルで quarantine 属性を削除する（推奨）**

```bash
xattr -cr /Applications/Gas\ Mask.app
```

**方法 2: 右クリックから開く**

Finder で `Gas Mask.app` を右クリック（または Control+クリック）→「開く」を選択します。
