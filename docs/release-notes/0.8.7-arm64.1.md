# Gas Mask 0.8.7-arm64.1

このリリースは、更新が止まっている本家 Gas Mask をベースにした Apple Silicon 向けの継続メンテナンス版です。

## 変更点

- 管理者権限の認証時に Touch ID を利用可能に（Touch ID 非搭載 Mac では従来通りパスワード入力にフォールバック）
- hosts ファイル一覧のドラッグ＆ドロップによる並び替えに対応（順序はアプリ再起動後も保持）
- Sparkle によるアプリ内自動更新を有効化
- 日本語ローカライズの改善（Preferences タブ名、ステータスバー表示など）

## 動作環境

- macOS 13 Ventura 以降
- Apple Silicon Mac

## インストール

### 「壊れているため開けません」と表示される場合

本アプリは Apple Developer ID による署名・公証を行っていないため、macOS Gatekeeper によってブロックされることがあります。

**方法 1: ターミナルで quarantine 属性を削除する（推奨）**

```bash
xattr -cr /Applications/Gas\ Mask.app
```

**方法 2: 右クリックから開く**

Finder で `Gas Mask.app` を右クリック（または Control+クリック）→「開く」を選択します。

## 既知事項

- notarization は自動化していません。初回起動時に Gatekeeper の表示が出る場合は上記の手順で対処してください。
