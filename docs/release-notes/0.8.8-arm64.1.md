# Gas Mask 0.8.8-arm64.1

## 変更点

- hostsメニューを"-"区切りで階層サブメニューに自動グループ化
- 「Hosts変更を反映」サブメニューを追加（DNSキャッシュクリア・Chromeソケットプールフラッシュ）
- hostsファイルごとにカスタムステータスバーアイコンを設定可能に
- PrivilegedActionsからbiometric認証パスとカスタムプロンプトを削除し簡素化
- CIマトリクスからmacOS 26 / Xcode 26.2を除外
- コード署名フラグをCODE_SIGN_IDENTITY=-に統一

## 動作環境

- macOS 13 Ventura 以降
- Apple Silicon Mac

## インストール

### 「壊れているため開けません」と表示される場合

本アプリは Apple Developer ID による署名・公証を行っていないため、macOS Gatekeeper によってブロックされることがあります。

**方法 1: ターミナルで quarantine 属性を削除する（推奨）**

```bash
xattr -cr /Applications/Gas\ Mask.app
```

**方法 2: 右クリックから開く**

Finder で `Gas Mask.app` を右クリック（または Control+クリック）→「開く」を選択します。
